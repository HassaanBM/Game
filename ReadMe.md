# Retro Arcade Mini Game 👾

Hey! Thanks for stopping by.

This little game exists because I spent some time exploring different ideas for my website. I tried a bunch of things, had fun experimenting, and in the end, this tiny retro arcade game was the one I decided to keep.

There's not much to learn—just tap or click to jump.

That's it.

The whole point is to give anyone who stumbles across my website something fun to play with while they're wandering around. If you get a little lost, you might as well enjoy the journey.

## Why so simple?

Because it was meant to be.

I wanted something that was easy to pick up, a little nostalgic, and just fun enough to make someone smile for a minute or two.

## Will it change?

Maybe.

I like revisiting old ideas, so this project might get a few updates here and there whenever inspiration strikes. Or it might stay exactly as it is—a tiny snapshot of an idea that made me curious enough to build it.

## Just a small piece of something bigger

This isn't the main project I'm working on—it's more like a little side quest. A fun detour while the bigger adventure is still under construction.

If you found this repository, thanks for checking it out. I hope you have as much fun playing it as I had building it. 🎮
